\begin{center}
\textbf{\large{PSCyr~--- коллекция русифицированных PostScript-шрифтов \\
в формате Type1 \\}}
\end{center}

\bigskip
В настоящее время получили распространение два пакета, позволяющие
использовать векторные (Type1) варианты базовых \TeX'овских шрифтов
(Computer Modern) для набора русских текстов: \texttt{cmcyr} и \texttt{wncyr}.
Работа с пакетом PSCyr (версии 0.4) позволяет получить доступ ещё к следующим
\textit{четырнадцати} семействам шрифтовых гарнитур в формате Type1. \\

\textbf{Шрифты с засечками:}

\begin{itemize}
\item \textac{Академическая (AcademyPSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}}
\item \textaq{Антиква (AntiquaPSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}, \textbf{\textit{Bold Italic}}}
\item \textha{Балтика (HandbookPSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}}
\item \textco{Бодони (CollegePSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}}
\item \textjn{Журнальная (JournalPSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}}
\item \textlz{Лазурская (Lazurski), начертание Regular}
\item \texttm{Таймс (TimesNewRomanPSMT), начертания Regular, \textbf{Bold},
\textit{Italic}, \textit{\textbf{Bold Italic}}}
\end{itemize}

\textbf{Рубленые шрифты:}

\begin{itemize}
\item \textar{Ариал (ArialMT), начертания Regular, \textbf{Bold},
\textit{Italic}, \textit{\textbf{Bold Italic}},
{\fontseries{eb}\selectfont Black}}
\item \texttx{Букварная (TextbookPSCyr), начертания Regular, \textbf{Bold},
\textit{Italic}}
\item \textma{Журнальная Рубленая (MagazinePSCyr), начертания Regular,
\textbf{Bold}, \textsl{Italic}}
\end{itemize}

\textbf{Моноширинные шрифты:}

\begin{itemize}
\item \textcr{Курьер (CourierNewPSMT), четыре начертания: Regular,\\
\textbf{Bold}, \textit{Italic}, \textit{\textbf{Bold Italic}}}
\item \texter{ER Курьер (ERKurierPSCyr), четыре начертания: Regular,\\
\textbf{Bold}, \textit{Italic}, \textit{\textbf{Bold Italic}}}
\end{itemize}

\textbf{Декоративные шрифты:}

\begin{itemize}
\item \textcp{Кладезь (CooperPSCyr), одно начертание}
\item \textad{Рекламный (AdvertisementPSCyr), одно начертание}
\end{itemize}

Пакет позволяет пользоваться стандартными для \LaTeXe\ командами смены
шрифтов (\verb"\tiny"$\dots$\verb"\Huge") и смены начертания
(\verb"\textbf"$\dots$\verb"\textrm").

\bigskip

Примеры: \textsc{капитель Антиква}, \tmfamily\textsl{наклонный и}
\textbf{полужирный Таймс}, \arfamily\textsl{\large большой наклонный
Ариал}, \mafamily{\tiny малюсенькая Журнальная рубленая,}
\txfamily\textbf{\small маленькая полужирная Букварная}.
